# Namaste Parking

## Getting Started

First, run the development server:

```bash
# install
pnpm install
# start
pnpm dev
# test
pnpm test
```

Open [http://localhost:8000](http://localhost:8000) with your browser to see it in action.

## Choices

- Next.js
- React
- TypeScript.
- CSS Modules.
- Validation - zod.
